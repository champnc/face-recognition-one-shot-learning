- Manual -

การติดตั้ง library
1. ใช้คำสั่ง 
    - pip install requirements.txt

การเก็บภาพ
1. ใช้คำสั่ง
    - python add_data.py --face_name ชื่อโฟลเดอร์
2. โปรแกรมจะเริ่มทำการเก็บภาพใบหน้า
3. เมื่อเก็บไปสักระยะ กด Q เพื่อทำการออกโปรแกรม
4. ภาพที่ถ่ายจะอยู่ใน folder images/ชื่อโฟลเดอร์

การ train model
1. ใช้คำสั่ง
    - python train_classifier.py
2. เมื่อรันคำสั่งจะได้ไฟล์ my_classifier.pkl

ระบบทำงานหลัก
1. ใช้คำสั่ง
    - python main.py
2. เมื่อรันคำสั่งโปรแกรมจะเริ่ม detect ใบหน้า หากเจอหน้าที่มีอยู่แล้วจะทำการเพิ่มเข้าไปในโฟลเดอร์เดิม หากพบเป็น unknown จะเพิ่มลงในโฟลเดอร์ unknown
3. โปรแกรมรับคำสั่ง 3 ปุ่ม
    - ปุ่ม A นำภาพในโฟลเดอร์ unknown ไป retrain
    - ปุ่ม Q ออกจากโปรแกรม
    - ปุ่ม C ลบภาพทั้งหมดในโฟลเดอร์ unknown
